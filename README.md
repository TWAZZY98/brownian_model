## Install-linux ![Python](https://img.shields.io/badge/Python-3.8+-blue)

1) git clone [repo]<br>
2) cd brownian_model<br>
3) Create a python virtual enviroment to install the packages<br>
  - sudo apt install python3-venv<br>
  - python3 -m venv venv<br>
  - source venv/bin/activate<br>
    - you will know that you are in the virtual enviroment when (venv) appears<br>
  - pip install -r requirements.txt<br>
    - installs all required packages<br>
4) cd app<br>
5) python main.py<br>
